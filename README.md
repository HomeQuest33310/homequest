# HomeQuest

Transformez les tâches ménagères en aventures coopératives pour toute la famille.

HomeQuest est un RPG familial open source où les parents, enfants et invités deviennent des aventuriers. Les corvées deviennent des quêtes, la maison devient un royaume, et toute la famille combat des boss ensemble.

## Valeurs

- Open source
- Gratuit
- Sans publicité
- Sans achats intégrés
- Respectueux de la vie privée
- Pensé pour les familles
- Auto-hébergeable

## Stack technique

- Flutter
- Supabase
- PostgreSQL
- Firebase Cloud Messaging
- GitHub

## Structure

```text
app/              Application Flutter
supabase/         Migrations, fonctions et données initiales
content-packs/    Campagnes et contenus du jeu
docs/             Documentation projet
assets/           Illustrations, icônes et audio
.github/          Templates et automatisations GitHub
```

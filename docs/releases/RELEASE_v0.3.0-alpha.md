# Release v0.3.0-alpha

## Objectif

Stabiliser la base jouable de HomeQuest avant d'ajouter les mécaniques avancées.

## Inclus

- Dashboard du Royaume stabilisé.
- Correction du bug de layout `FilledButton` dans le registre des quêtes.
- Vision produit ajoutée à la racine du projet.
- Préparation du projet pour les prochains modules : quêtes terminables, XP, or, compétences, boss.

## Mise à jour

Dézipper cette release à la racine du dépôt, accepter les remplacements, puis relancer l'application depuis `app/`.

# Contributing to HomeQuest

First of all, thank you for your interest in HomeQuest! ❤️

Our goal is to build the best open-source cooperative family RPG that transforms household chores into adventures.

## Our Values

- Family first.
- Cooperation over competition.
- Respect for children's privacy.
- Accessibility for everyone.
- No advertisements.
- No pay-to-win.
- Open Source by default.

## Ways to Contribute

You can help by reporting bugs, suggesting features, designing illustrations, improving UI, writing stories and quests, translating, testing, or improving documentation.

## Before You Start

1. Search existing issues before creating a new one.
2. Open a discussion for large features before coding.
3. Keep pull requests focused on a single topic.

## Development Workflow

1. Fork the repository.
2. Create a feature branch.
3. Commit your changes with clear commit messages.
4. Open a Pull Request.
5. Participate in code review.

## Game Design Philosophy

Never add a feature that shames children, encourages unhealthy competition, requires spending money to progress, or collects unnecessary personal data.

> Does this make family life more enjoyable?

If the answer is yes, you are probably moving in the right direction.
